## Installation

Use the package manager [pnpm](https://pnpm.io/installation) project dependencies.

```bash
pnpm install
```

## Usage

```bash
# start dev server
pnpm run dev

# build for production
pnpm run build

# test the prod build
pnpm run serve

# deploy to github pages
pnpm run deploy
```

You can check other scripts in `package.json` of the root repo and each package
